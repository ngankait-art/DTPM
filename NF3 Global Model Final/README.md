# Ar/NF₃ ICP Global Plasma Model

A zero-dimensional (0D) global model for an argon/nitrogen trifluoride inductively
coupled plasma (ICP) discharge, reproducing the framework of:

> **Huang et al. (2026)**  
> *"Study for Ar/NF₃ plasma chemistries in inductively coupled discharges  
> with global model and experiment"*  
> Plasma Sources Sci. Technol. **35**, 015019  
> DOI: [10.1088/1361-6595/ae35a0](https://doi.org/10.1088/1361-6595/ae35a0)

---

## Overview

The model solves time-dependent ordinary differential equations (ODEs) for 21
plasma species densities and the electron temperature, integrating until steady
state is reached. It includes 155 volumetric reactions, surface recombination,
gas pumping, and a self-consistent electron energy balance.

### Key features

- **21 species**: 10 neutrals (NF₃, NF₂, NF, F₂, N₂, F, N, Ar, Ar(1s5),
  Ar(4p)), 8 positive ions (NF₃⁺, NF₂⁺, NF⁺, F₂⁺, N₂⁺, F⁺, N⁺, Ar⁺),
  2 negative ions (F⁻, F₂⁻), and electrons.
- **155 gas-phase reactions**: electron-impact ionization, dissociation,
  attachment, excitation; Penning dissociation/ionization; neutral–neutral
  recombination; ion–neutral charge exchange; ion–ion neutralization;
  high-temperature thermal dissociation; three-body recombination.
- **Surface recombination**: species-dependent sticking coefficients
  (γ_F = 0.01, γ_N = 0.07, γ_Ar* = 1.0).
- **Ion wall losses**: Thorsteinsson–Gudmundsson formulation with
  electronegativity corrections for modified Bohm velocity, ambipolar
  diffusion, and edge-to-center density ratios.
- **Power balance**: self-consistent electron energy equation with collisional
  and wall energy losses.
- **Gas temperature**: empirical correlation from Thorsteinsson (2010).
- **Pumping**: constant-flow model with species-dependent pump-in and pump-out.

### Validated against

- Electron density vs. power, pressure, and NF₃ ratio (cutoff probe data)
- F/Ar density ratio from OES actinometry
- NF₃ dissociation fraction
- Ion composition (dominant species identification)
- Electronegativity trends

## Quick Start

### 1. Run the base case

```bash
python solver.py
```

This runs a single simulation at the default base-case conditions (80/20 Ar/NF₃,
30 mTorr, 600 W) and prints a full summary of steady-state results including
all species densities, electron temperature, gas temperature, dissociation
fraction, and electronegativity.

**Expected output** (abbreviated):

```
Solving NF3/Ar global model...
  Conditions: 80.0/20.0 sccm Ar/NF3, 30.0 mTorr, 600.0 W
  Chamber: R=21 cm, L=6 cm, V=8312.7 cm^3
  Gas temperature: 1158 K
  Solution converged (1529 function evaluations)

============================================================
STEADY-STATE RESULTS
============================================================
Electron temperature: 2.40 eV
Electron density: 1.200e+17 m^-3 (12.00 x10^16 m^-3)
NF3 dissociation: 98.4%
Electronegativity (n_neg/n_e): 0.354
```

**Runtime**: ~3 seconds per condition on a modern laptop.

### 2. Run parametric sweeps

```bash
python sweep.py
```

This runs power sweeps (500–1000 W at 30 mTorr) and pressure sweeps
(10–100 mTorr at 600 W), then generates a 6-panel summary figure saved as
`nf3_results.png`.

**Runtime**: ~40 seconds (12 conditions total).

**Output**: `nf3_results.png` — 6-panel plot showing:
- (a) Electron density vs. power
- (b) Electron density vs. pressure
- (c) Neutral species densities vs. pressure (log scale)
- (d) Electron temperature and gas temperature vs. pressure
- (e) Electronegativity vs. pressure
- (f) NF₃ dissociation vs. pressure

### 3. Generate publication-quality figures

```bash
python gen_figures.py
```

Generates 8 individual PDF figures in `figures/`, suitable for inclusion in the
LaTeX report. These are the same parametric sweeps as `sweep.py` but rendered
as separate, high-resolution PDF files with serif fonts.

**Runtime**: ~40 seconds.

### 4. Compile the LaTeX report

```bash
pdflatex report.tex
pdflatex report.tex
```

Run twice to resolve cross-references and table of contents. Produces
`report.pdf` (16 pages) containing the full mathematical formulation,
governing equations, reactor parameters, numerical methods, results with
annotated figures, and discussion.

**Note**: The figures in `figures/` must be generated before compiling the
report (step 3).

## Customizing Simulations

### Changing operating conditions

Edit the `ReactorConfig` constructor parameters in `solver.py` (at the bottom,
in the `if __name__ == '__main__'` block):

```python
config = ReactorConfig(
    R=0.21,              # Chamber radius [m]
    L=0.06,              # Chamber height [m]
    P_abs=600.0,         # Absorbed RF power [W]
    pressure_mTorr=30.0, # Operating pressure [mTorr]
    flow_Ar_sccm=80.0,   # Argon flow rate [sccm]
    flow_NF3_sccm=20.0,  # NF3 flow rate [sccm]
    Twall=300.0,         # Wall temperature [K]
)
```

### Running a custom sweep

Use the `NF3GlobalModel` class directly in a Python script:

```python
from solver import NF3GlobalModel, ReactorConfig, gas_temperature

# Define conditions
config = ReactorConfig(P_abs=800.0, pressure_mTorr=50.0,
                       flow_Ar_sccm=60.0, flow_NF3_sccm=40.0)

# Create and solve model
model = NF3GlobalModel(config)
sol = model.solve(t_end=0.05, n_points=500)

# Extract steady-state results
y_ss = sol.y[:, -1]
Te = y_ss[model.idx_Te]

# Get species densities
for species, idx in model.species_idx.items():
    n = max(y_ss[idx], 0)
    if n > 1e14:
        print(f"  {species:8s}: {n:.3e} m^-3")
```

### Adding or modifying reactions

Edit `reactions.py`. Each reaction is a dictionary appended to the `REACTIONS`
list:

```python
REACTIONS.append({
    'id': 156,                                    # Unique integer ID
    'type': 'electron',                           # Reaction type (see below)
    'reactants': ['e', 'NF3'],                    # Reactant species list
    'products': ['NF2', 'F', 'e'],                # Product species list
    'rate_coeff': arrhenius_Te(A, B, C, D=1.0),   # Rate coefficient function
    'energy_loss': 2.5,                           # Electron energy loss [eV]
    'description': 'e + NF3 -> NF2 + F + e'       # Human-readable label
})
```

**Reaction types**: `'electron'`, `'penning_diss'`, `'penning_ion'`,
`'neutral'`, `'ion_neutral'`, `'ion_ion'`, `'detachment'`, `'high_temp'`,
`'three_body'`.

**Rate coefficient forms**:

- `arrhenius_Te(A, B, C, D=1.0)` — returns `k(Te) = A * Te^B * exp(-C / Te^D)`
  where `Te` is in eV.
- For reactions with `Te` in Kelvin (e.g., R54–R57), define a custom function:
  ```python
  def _k(Te, Tg=300.0, Tng=1.0):
      Te_K = Te * 11604.5
      return A * Te_K**B * np.exp(-Ea_K / Te_K)
  ```
- For heavy-particle reactions, use `Tg` and `Tng = Tg/300`:
  ```python
  def _k(Te, Tg=300.0, Tng=1.0):
      return A * Tng**B * np.exp(-Ea / Tg)
  ```

## Model Equations Summary

The model solves the following coupled system at steady state:

| Equation | Description | Reference |
|----------|-------------|-----------|
| dn_X/dt = Σν·R − k_wall·n − pump | Particle balance for each species | Eq. 5 in paper |
| n_e = Σn⁺ − Σn⁻ | Charge neutrality | Eq. 6 |
| k_wall = [Λ²/D + 2V(2−γ)/(Avγ)]⁻¹ | Neutral wall loss | Eq. 11 |
| dn_i/dt|_wall = −u_B · A_eff / V · n_i | Ion wall loss (Bohm) | Eqs. 15–22 |
| d(3/2·e·n_e·Te)/dt = P/V − losses | Power balance | Eq. 25 |
| T_g = 300 + s(p)·log₁₀(P/40) | Gas temperature | Eqs. 8–9 |

See `report.pdf` for the complete mathematical derivation with all sub-equations.

## Troubleshooting

### Solver fails to converge

- Increase `t_end` in `model.solve(t_end=0.05)` to allow more time for steady
  state (try 0.1 or 0.2).
- Check that the operating conditions are physically reasonable (pressure
  > 1 mTorr, power > 50 W).
- Reduce the initial electron temperature: change `y0[self.idx_Te] = 4.0` to
  `3.0` in `solver.py`.

### Negative densities in output

The solver clamps densities to zero at each RHS evaluation. If negative values
appear in the final output, increase `atol` in the `solve_ivp` call (currently
`1e6`; try `1e8`).

### Matplotlib backend errors

If running on a headless server without a display:

```bash
MPLBACKEND=Agg python sweep.py
```

All scripts already set `matplotlib.use('Agg')` so this should not be needed.

### LaTeX compilation fails

Ensure the `figures/` directory is populated before running `pdflatex`. The
`report.tex` file includes `\includegraphics` calls to files in `figures/`.

## Known Limitations

1. **Maxwellian EEDF assumed**: Rate coefficients are integrated over a
   Maxwellian electron energy distribution. Non-Maxwellian effects (which can
   be significant in electronegative plasmas) are not captured.

2. **0D volume-averaging**: No spatial resolution. Edge effects, skin depth,
   and transport are captured only through effective h-factors.

3. **No O₂ chemistry**: The model covers pure Ar/NF₃ mixtures only. Extension
   to Ar/NF₃/O₂ would require additional species (O, O₂, NO, N₂O, etc.) and
   ~150 additional reactions.

4. **Empirical gas temperature**: T_g is not self-consistently computed from a
   neutral energy balance but uses a fitted correlation.

5. **Steady-state only**: The ODE integrator runs to steady state. Pulsed
   plasma or time-resolved dynamics would require modifications to the pumping
   and power input terms.
