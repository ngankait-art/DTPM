# SF₆ ICP Global Plasma Model

Zero-dimensional global model for SF₆ inductively coupled plasma (ICP) discharges, reproducing the results of Kokkoris et al., *J. Phys. D: Appl. Phys.* **42**, 055209 (2009).

## Overview

The model solves 18 coupled ODEs (13 species densities, electron temperature, 4 surface coverages) from pre-discharge initial conditions to steady state using BDF time integration. It includes:

- **50 gas-phase reactions** (dissociation, ionisation, attachment, recombination, detachment)
- **40 surface reactions** (adsorption, recombination, sputtering, deposition)
- **Druyvesteyn EEDF** rate coefficients (Table 1 of Kokkoris et al.)
- **Troe fall-off** pressure-dependent recombination (Ryan & Plumb 1990)
- **Chantry** neutral wall loss model
- **Two h-factor formulations** for electronegative ion wall loss

## Model Versions

| Parameter | V1 | V3 |
|---|---|---|
| h-factor | Lee & Lieberman (1995) | Kim et al. (2006) / Thorsteinsson (2010) |
| η (power coupling) | 0.50 | 0.70 |
| s_dep (deposition probability) | 0.03 | 0.10 |

V3 resolves three qualitative discrepancies present in V1: the pressure rise plateau, the reversed F density trend, and the flat SF₅⁺ profile in the pressure sweep.

## Quick Start

```bash
# Install dependencies
pip install numpy scipy matplotlib

# Run both model versions (takes ~5 minutes)
python run_model.py both

# Generate overlay plots
python plot_overlays.py
```

Results are saved to `output/` (CSV) and `output/figures/` (PNG).

To run only one version:
```bash
python run_model.py v1    # Original parameters
python run_model.py v3    # Updated h-factor and parameters
```

## Kokkoris Extracted Data

All paper data was extracted by the researcher using [WebPlotDigitizer](https://automeris.io/WebPlotDigitizer) with calibrated axes. Each CSV has two columns with a header:

```
x,y
18.73,179087628505506900000
102.81,159554035094438000000
...
```

File naming convention: `kokkoris_[quantity]_vs_[sweep].csv`

- `_vs_power`: Power sweep (x = Power in W, p_OFF = 0.921 Pa)
- `_vs_poff`: Pressure sweep (x = p_OFF in Pa, P = 2000 W)
- `_model_`: Paper's "basic" model curve (Figs 5–6 only)
- `_nosurface_`: Paper's "no surface reactions" configuration
- `_nodeposition_`: Paper's "no SFx deposition" configuration
- `_exp_`: Experimental data points
